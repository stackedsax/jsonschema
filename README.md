JsonSchemaCLI
=======

JSON Schema Generator for the command line.


Example
=======

node Schema.js '{"address":{"streetAddress":"21 2nd Street","city":"New York"},"phoneNumber":[{"type":"home","number":"212 555-1234"}]}'
